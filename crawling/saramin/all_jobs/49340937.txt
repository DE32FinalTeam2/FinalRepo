백엔드 개발자(JAVA) 채용
서비스 소개
이파피루스는? 풍부한 경험과 탄탄한 기술력을 바탕으로 안정적이고 효율적인 전자문서 환경을 실현하는 문서AI 솔루션 전문기업입니다. 

■ 웹기반 문서 인프라 기업
 - 문서 라이프사이클을 아우르는 웹 기반 문서 인프라를 제공합니다
 - 누구나 쉽게 이용할 수 있는 스마트 문서 환경을 실현합니다
 - 국세청, 대법원 등 국내 주요 공공기관 및 네이버, 포스코, KB증권 등 대기업 업무 시스템에 사용되고 있습니다

■ 해외 진출과 사업 확장
 - 독보적인 문서 기술 및 노화우를 보유한 유수의 해외 자회사들과 함께합니다
 - Open AI의 "ChatGPT"에 PDF 솔루션 기술을 공급하고 있습니다
 - 인스탁스 시리즈 회사 "후지필름"에 고성능 PDL 엔진 'ghostscript' 기술을 공급하고 있습니다
 - AI 문서 질의 응답 서비스 'Stream Docs.ai' 출시하여 편의성 및 신뢰성 제공과 AI 환각 문제까지 해결한 혁신적인 서비스를 제공하고 있습니다.

■ 스마트 공장을 위한 예지보전
 - AI, 빅데이터, 클라우드, IoT 기술을 결합한 예지보전 솔루션을 제공합니다
 - 산업용 장비·설비 이상을 미리 감지하고 돌발 고장을 예방합니다
 - 삼성전자, 한화큐셀, 현대케피코, CJ제일제당 등 산업 현장 전 분야에서 활약합니다

이파피루스는 'Document Infrastructure for the Web'이라는 비전 아래 글로벌 전자문서 인프라 플랫폼을 구축하기 위한 역량확보에 집중하고 있습니다.
이파피루스의 새로운 비전과 함께하며, 즐겁게 도전하고 싶은 분들께 포지션 제안 드립니다.

사용 기술
Spring Framework REST API AWS Git Jenkins JSP Spring Java
주요업무
• 신규 제품 기획 및 개발
• 기존 제품군 기능 개선 및 추가
• 기술자료 작성 및 관리
자격요건
• Java Application 개발 경험 3-15년
• Spring framework 3.x 이상의 환경에서의 API 설계 경험 보유자
• 스스로 데이터베이스를 설계하고 개발/운영 경험 보유자
• 2~3명 이상 규모의 개발 프로젝트 리딩 경험 보유자
우대사항
• Java/Spring/자료구조/스레드 프로그래밍에 대한 깊은 이해도
• Spring Boot 및 Spring Cloud 어플리케이션 사용 경험
• 객체지향적 사고방식
• 큰 어려움 없이 해당 분야 영문 기술자료를 이해할 수 있는 수준의 영어 역량
• Jenkins 기반의 CI/CD 활용 경험
• Git/Git flow/AWS 사용 경험
• RESTful API 설계 또는 작성 경험
• '읽기 좋은 코드가 좋은 코드'라는 가치관 보유자
• Java 1.6 이상의 JVM thread dump 분석 가능
• 1개 이상의 Message Queue 기술 실무 경험
• 대용량 분산처리 경험
• 코딩역량인증(PCCP, PCCE) 인증서 보유자 우대
마감일 및 근무지
• 마감일 : ~2024-12-05
• 근무지
- 경기 성남시 분당구 판교역로 230, 비동4층 408,409,410호
복지 및 혜택
[가치제안]
• 다양한 파트너사와 협업하여 원스톱 서비스를 지원하기에 뛰어난 문제해결능력 및 커뮤니케이션능력을 향상시킬 수 있습니다.
• 고객 서비스에서 나아가 파트너 교육, 클라우드 운영 관리 확대로 본인의 역량을 키울 수 있습니다.
• 독립적으로 업무를 수행하여 솔루션엔지니어 직무의 전반적인 프로세스를 습득할 수 있습니다.

[근무환경]
• 수평적인 조직문화, 복장/연차(시간단위) "완전" 자유
• 불필요한 야근/회식 강요없음
• 최고수준 개발장비, 풍족한 간식/음료 제공

[다양한 지원제도]
• 전월세 자금 대출, 암/실손보험 가입
• 내일채움공제 가입, 퇴직연금제도 시행
• 식대 지원, 경조사비/휴가 지원
• 사내 동호회 지원
