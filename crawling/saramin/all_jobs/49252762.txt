로봇 개발자
서비스 소개
KALMAN은 로봇이 우리의 삶과 일상을 변화 시킬 것이라는 확신으로, 훌륭한 팀원들과 함께 업무를 수행하고 있습니다. 

자체 개발하는 로봇 솔루션을 기반으로 기존에 어려웠던 작업들을 로봇을 통해 가능하도록 만드는 새로운 개념의 로봇 서비스를 제공합니다. 

현재, 저희 팀에서 개발하는 고성능 로봇들은 원자력 발전소와 같이 신뢰성을 요구하는 분야에서 적용되고 있습니다.
이 포지션의 AI 예상 면접 질문이 궁금하다면 점핏에서 확인해보세요!
모집부문 / 상세내용
사용 기술
MCU MATLAB Python HTML5 C C++ Embedded HW ROS Rust
주요업무
저희 팀과 함께 로봇을 개발하고자 하는 분들을 상시 모십니다. 
병역특례 지원 가능합니다.
주요 업무는 크게 아래와 같이 나눌 수 있습니다.

• Mechanical Engineer : 로봇 하드웨어 설계
- Gear/Leg/Arm/Actuator 설계 
- Tool : Inventor, Fusion360, Solidworks, AutoCAD 등 CAD 관련 업무

• Control Engineer : 로봇 제어기 설계 및 시뮬레이션
- PID, Optimal Control, Reinforcement learning, Filter 설계
- MPC, 모방학습 및 딥러닝 활용 제어(sim2real) 엔지니어링
- C/C++, Python, MATLAB 등 프로그래밍

• Software Engineer : 로봇 응용 프로그램 / 서비스 개발
- 영상처리, Deep-learning
- 다중 카메라 데이터 취득 및 처리
- QT, HTML
- C/C++, Python, Java, Rust 등 프로그래밍 

• Embedded Engineer : 로봇 컴포넌트 개발
- STM32, Arduino, DSP 등의 MCU를 사용한 제품 설계 
- Circuit Design/Artwork 
- C/C++, Python 등 프로그래밍

• Robot Operator : 로봇 설치/조종/서비스 (경험자 우대)
- 로봇 혹은 RC 조종 유경험자
- 드론, 모바일, 수중 드론 등
- 고객 의견 반영, 로봇 사용성 개선
자격요건
• 4년제 대학(공학계열) 졸업 이상
• 포트폴리오
우대사항
• 고성능 엑추에이터 설계 분야 경험
• 로봇 관련 서비스 제작 경험
• 로봇 제작 경험
• 로보틱스 관련 연구실 졸업자
• [고성능 필드 로봇 개발]에 관심을 갖고 있다면 저희 회사에 필요한 분입니다.
마감일 및 근무지
• 마감일 : ~2024-11-21
• 근무지
- 경기 안양시 동안구 엘에스로 142 안양 skv1 센터 407호
복지 및 혜택
• 식대 제공
• 자기차량으로 출퇴근 가능 (입사 선착순)
• 적합한 TOOL 제공/권장
• 유연한 출근시간 : 8시~11시, 유연하게 선택 가능
• 쾌적한 연구실과 실험실 : 연구실 2개동/실험실 2개동
• 짧은 회의시간 : 1시간 이내/1주
• 자유로운 연구 문화 : 주로 Notion/Teams를 통한 업무 공유
• 불필요한 회식 없음
채용절차 및 기타 지원 유의사항
• 서류전형 > 1차 서류 면접 > 2차 대면 면접 > 최종합격
• 포트폴리오를 함께 제출해 주세요. 프로젝트에서 본인이 참여한 부분에 대해서 명시를 해주시면 좋습니다.
*입사지원 서류에 허위사실이 발견될 경우, 채용 확정 이후라도 채용이 취소될 수 있습니다
