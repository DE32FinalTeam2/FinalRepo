Telit Cinterion: Who we are

Telit Cinterion은 기계와 기업 비즈니스 시스템 및 데이터베이스 간의 원활하고 안전한 엔드 투 엔드 연결 및 통합을 위한 제품 및

서비스 개발 및 배포의 선구자입니다. Telit Cinterion이 보유한 업계 최고의 deviceWISE 및 secureWISE 플랫폼은 인텔리전스 활성화,

비용 절감, 수익 증대 및 규정 준수 개선으로 인정받고 있습니다.

고객솔루션에는 텔레매틱스, 식음료점 장비, 산업 프로세스 및 도구, 청구시스템, 자동차, 트럭 장치, 컨테이너 추적, 콜드체인 모니터링

등과 같은 분야의 원격 기계 모니터링 및 제어가 포함됩니다.

Telit Cinterion은 풍부한 경험, 광범위한 프로그래밍 언어/환경에 대한 관심, 적응력, 빠른 학습 력 및 탁월한 문제해결 기술을 바탕 으

로 빠르게 변화하는 환경에서 근무할 제조 지향 기술 엔지니어/프로그래머를 찾고 있습니다. 주요 분야는 connected Edge device, 게이트

웨이, 제어 및 클라우드 기반 산업용 IoT 플랫폼 제품입니다.


About your team

우리 팀은 Embedded Linux 기반의 Software 에서 HW device를 구동 시키기 위한 가장 낮은 layer의 핵심적인 업무를 담당합니다. 

주 업무로는Bootloader, Linux Kernel, Device Drivers, Root File system, Board configuration files, Firmware updating tool 과 같은

영역에서 개발합니다. 

또한, 제품의 성능 향상과 안정성 확보를 위해서 깊이 있고 광범위한 연구활동을 통해 우리 제품이 시장에서 경쟁력 있는 품질을

확보하는데 기여하고 있습니다.


What you'll do

- 프로젝트에 최적화된 Linux build system 환경을 조상 및 관리

- Bootloader, Kernel 수정을 통해 boot-up performance 및 system stability 향상을 위한 연구 개발

- Peripheral interface 동작을 위해 Kernel layer의 Device driver를 개발하고 관리

- 고객의 다양한 요구로부터 customized 된 device driver를 개발하고 제공

- Customer field에서 수행되는 IOT module의 Firmware update engine과 tool을 개발하고 관리

- 고객 지원 팀과 협력하여 기술적 문제를 해결하고 사용자 피드백을 반영

 
You should have

- 컴퓨터 공학 및/또는 컴퓨터 과학 및/또는 전기 공학 및/또는 정보 기술 학사 학위

- 해당 분야 2년 ~ 10년 경험

- 리눅스 환경에서 C/C++을 사용한 프로그래밍 경험

- 문제 해결 능력과 분석적 사고

- 협력을 중시하는 팀 플레이어

- 해외 여행에 결격사유 없는 자

- 비즈니스 영어 소통 능력


Nice if you have

- 임베디드 리눅스 시스템에서 device driver 에 대한 개발 경험

- MCP 나 eMMC 메모리를 이용한 f/w update engine 설계나 개발 경험

- IOT module 의 application processor 위에서 개발 경험

- Qualcomm 칩셋 유경험자
