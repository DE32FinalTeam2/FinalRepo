사랑하는 반려동물을 위한 최고의 케어 서비스, 펫닥

포포즈 반려동물 장례식장, Vet화점 동물병원 용품 전문 플랫폼 등 반려동물을 위한 다양한 서비스를 제공하며 각 분야의 전문가들이 모여 빠른 속도로 성장하고 있는 기업입니다. 
저희 펫닥에 합류하셔서 함께 꿈을 펼치고 가치있는 서비스를 만들어 가실 분을 찾습니다.
이 포지션의 AI 예상 면접 질문이 궁금하다면 점핏에서 확인해보세요!

주요업무
[데이터 분석 엔지니어링]
• 데이터 과학자와 협업해 신규 BI 개발을 하고 사용자들의 피드백을 통해 지속적으로 BI를 고도화합니다. 
• 데이터를 사용하는 전사 조직에 맞춰 데이터 사용 및 활용 정책을 수립하고 가이드합니다. 
• 기존에 개발돼있는 Web Data Application 을 전사 조직과 협업해 고도화를 추진합니다. 
• 다양한 영업 채널에서 생성되는 데이터를 토대로, 데이터의 사용 영역의 확장 방안을 고민하고 아이디어를 실제로 구현합니다. 

[Data Architect]
• 데이터의 정합성, 일관성, 적시성을 위해 데이터 검증 로직을 개발하고 오류 최소화를 위해 기존 로직들의 고도화를 추진합니다. 
• 신규 BI 개발을 위해 전사 조직과 협업해 최적의 데이터 처리 로직을 개발합니다. 
• 업무 효율화를 위해 전사 데이터 표준을 정의하고 표준에 맞는 데이터 처리 로직을 개발합니다.

• Python을 능숙하게 사용할 수 있는 분 
• SQL을 통해 CRUD 작업을 능숙하게 하실 수 있는 분

• AWS, Azure 등의 클라우드 기반 개발 경험이 있는 분

• 전사 다양한 팀들과 열린 마음으로 의사소통해 최선의 문제해결 방안을 도출하고, 수립한 목표로 과제를 강력하게 추진할 수 있는 분 
• 데이터의 가치를 믿고, 실제로 데이터를 활용해 다양한 서비스를 만들어 보신 분
