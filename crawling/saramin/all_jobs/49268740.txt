컴퓨터 비전 딥러닝/머신러닝 개발자(ADAS/DMS/영상인식) (신입)
서비스 소개
에프에스솔루션은 영상 인식 기반 인공지능(AI) 알고리즘 설계에 특화한 테크 스타트업입니다. 
초경량 알고리즘 설계를 통해 하드웨어, 칩 제조사에 맞춤형 온디바이스(On-Device) AI 솔루션을 제공합니다. 
NPU뿐만 아니라 CPU 단에서도 효율적으로 동작하는 자체 구축 딥러닝 프레임워크(FS NNE 2.0)를 통해 비용 절감과 서비스 고도화 사이에서 고민하는 많은 전장 A/M, B/M 제조사 및 스마트 디바이스 제조사와 협업하고 있습니다.
이 포지션의 AI 예상 면접 질문이 궁금하다면 점핏에서 확인해보세요!
모집부문 / 상세내용
사용 기술
OpenCV C Docker PyTorch Python Embedded DeepLearning TensorFlow C++ MachineLearning
주요업무
• 딥러닝/머신러닝 모델 개발
- Classification(분류) / Object Detection(객체 검출)
- 데이터 셋 설계 및 구축 / 데이터 셋 통합 / Semi-Supervised Learning
- Image Augmentation / Auto-Labeling
- 딥러닝 모델 개발 및 경량화
• ADAS / DMS 개발
- ADAS : LDW / FCW / RCW / HMW / FVSA / MOD / FVSA / TSR / BSD / LPR 등
- DMS(Driver Monitoring Systems) : 운전자 신원인식 / 표정 / 시선 / 행위인식 등
• CCTV Surveillance
- 지능형 CCTV 솔루션 : 침입 / 배회 / 유기 / 폭행 / 화재 / 쓰러짐 / 신원인식
- 보행자 / 차량(차종/색상) / 번호판 인식/ 얼굴인식 / 인상착의(인종, 성별, 연령, 상하의, 신장)
• Embedded System
- 딥러닝 모델 변환 및 경량화(Quantization / Pruning)
- PC 기반에서 개발된 영상 인식 알고리즘 및 딥러닝 모델을 Embedded(CPU, NPU) 환경에 포팅 및 최적화
- Embedded System 환경에서 이미지 처리 및 노이즈 제거 등의 영상 인식 Algorithm 개발 및 포팅
