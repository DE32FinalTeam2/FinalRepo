모집부문 / 상세내용
사용 기술
CUDA RabbitMQ Git Docker Kafka REST API C++ C
주요업무
• CCTV RTSP 스트림을 받아 **NVIDIA DeepStream SDK**로 기본적인 영상 분석 기능을 적용
• **RabbitMQ**나 **Kafka**를 이용한 실시간 데이터 스트림 관리
• **CUDA**로 GPU 가속을 통해 효율적인 영상 처리를 지원
• **REST API**를 통해 시스템 간 간단한 데이터 연동을 지원
자격요건
• **C, C++ 언어 기반** 개발 경험
• 실시간 영상 스트리밍 프로토콜 **RTSP**에 대한 이해
• 영상 관련 프레임워크(ffmpeg,gstreamer) 활용 개발 경험
• RESTful API 설계 및 구현 경험
• **Docker**를 이용한 컨테이너화 및 배포 경험
• **RabbitMQ** 및 **Kafka**와 같은 메시지 브로커 사용 경험
• 문제 해결 능력 및 논리적 사고력
우대사항
• **TensorRT** 및 ONNX 생태계에 대한 심층적인 이해 및 경험
• **NVIDIA DeepStream SDK**를 활용한 영상 분석 시스템 개발 경험
• **CUDA**를 이용한 병렬 컴퓨팅 및 GPU 최적화 경험
• 자동화 도구 및 CI/CD 파이프라인 구축 경험
마감일 및 근무지
• 마감일 : ~2024-12-18
• 근무지
- 서울 서초구 태봉로 108 서울AI허브 304호 피아스페이스
