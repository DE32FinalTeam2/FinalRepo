Cloud Backend Engineer
서비스 소개
한화비전은 세계 최고 수준의 영상 기술을 기반으로 한 영상감시 외에 출입통제, 침임탐지 등 사업을 다각화하고 있습니다. 
1991년 보안 카메라 출시를 시작으로 약 30여 년간 영상보안 사업에 집중하여 국내 영상보안 시장 점유율 1위를 유지하고 있으며, 
글로벌 시장에서도 5,000개 이상의 네트워크를 구축, 전체 매출의 75%를 해외에서 거두며 활발한 영업 마케팅 활동을 펼치고 있습니다.
이 포지션의 AI 예상 면접 질문이 궁금하다면 점핏에서 확인해보세요!
모집부문 / 상세내용
사용 기술
PostgreSQL NestJS MQTT TypeScript MongoDB
주요업무
• 클라우드 서비스를 개발하는 agile scrum 팀에 소속되어 백엔드 개발자로서 업무를 수행해요.
• Cloud 기반 원격 장비 관리, AI metadata 기반 BI 솔루션, Cloud 기반 VMS 솔루션을 개발해요.
• Cloud 환경에서 IoT 장비로부터의 메시지 수집 및 처리, 대용량 데이터 처리, 영상 저장, 영상 스트리밍, 솔루션에 필요한 REST API 개발, 서비스간 통신 등의 백엔드 서비스를 개발해요.
자격요건
• 경력 4년 이상 백엔드 서비스 개발자
• DBMS 프로그래밍 가능: Postgresql, DocumentDB (혹은 MongoDB)
• Language: Typescript; Framework: Nestjs
• AWS ISaaS 및 PaaS 사용 경험 (ELB, EC2, S3, ELB, EC2, etc.)
• CI/CD 경험 및 git, github 사용 가능
• 메세징 미들웨어 연동 경험자 MQTT broker, Apache Kafka
• Cloud 웹 서비스 기술 이해
: RESTful, Micro-service, Open API, OAuth2/OIDC, JWT, PKI, RBAC(AuthZ), Secure-coding

※ 관련 포트폴리오가 있으시다면 제출 요청드립니다 ※
• 깃헙, 노션 등 프로젝트가 정리된 자료
• 이 외 자신을 나타낼 수 있는 모든 기술 자료 제출 가능
우대사항
• Agile scrum 경험자
• 미디어 스트리밍 프로토콜 및 코덱 이해 : RTSP/RTP, HLS, mp4, H.264/H.265 codec, WebRTC
• 영어로 의사 소통이 가능한 분

복지 및 혜택
’가족’을 생각하는 문화
다양한 복지제도와 행사를 통하여 건강한 가정을 지원합니다.
• 가족 초청 행사, 자녀 학자금 지원 제도, 아빠 휴가, 자녀 의료비 지원, 난임 시술 지원, 상조서비스 및 경조금/경조 휴가 지원, 본인/배우자 건강검진, 실비보험 등

• 동호회 지원, 사내 스터디 지원, 장기근속 포상, 멘토-멘티제도 운영 등
