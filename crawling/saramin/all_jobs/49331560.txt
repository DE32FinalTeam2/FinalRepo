이런 분을 찾습니다
대한민국 대표 온라인 명품 커머스 플랫폼 머스트잇은 2011년 설립 이후 치열한 경쟁 환경 속에서 ‘진짜는 모두가 알아본다’는 CEO의 경영철학 하에 신뢰와 정직을 바탕으로 고객과 판매자 모두에게 진정성을 주며 업계를 선도하고 있습니다. 
우리는 급변하는 시장 환경에 빠르게 대응하며 주도권을 잡고, 또 한 번 폭발적인 성장과 임팩트, 마법 같은 반등을 이루기 위해 치열하게 달리고 있으며, 이에 맞춰 결속력이 강한 조직을 만들기 위해 조직을 슬림하게 재편하며 밀도 높은 조직을 구성했고, 딱딱한 리더십이 아닌 성숙한 오너십을 가진 동료들과 함께 견고한 "One Team"을 구축했습니다. 
최고의 인재들은 서로를 통해 성장하고, 더 많이 더 빠르게 성과를 창출할 수 있다고 합니다. 이에, 우리와 함께 시장의 문제를 해결할 비범한 동료를 찾습니다.
[ MUST IT Development Environment ]
- Project/Issue Management Tool : Jira
- Knowledge Base : Confluence
- Source Management : Git (Github)
- IDE Tools : VSCode, IntelliJ, Xcode, Android Studio
- Deployment Tool : Jenkins
- Monitoring Tool : AWS CloudWatch, Grafana + Prometheus
- Logging Tool : AWS Athena, ElasticSearch + FileBeat/Logstash + Kibana

[ MUST IT Technology Stacks ]
- Mustit-service, Admin : PHP, CodeIgniter(2 or 3), HTML/CSS/javascript
- FrontWeb: Vue.JS + Typescript
- iOS App: iOS SDK + Swift
- Android App: Android SDK + Kotlin
- Auth/Facade API : Node.JS(v12+) + Koa + Typescript
- Biz API : SpringBoot2 + JPA + Kotlin
- Infra : AWS Services, AWS EKS(Kubernetes 1.23), Docker, Terraform

      - 머스트잇의 기존 커머스 레거시 서비스 마이그레이션머스트잇 서비스 개발
