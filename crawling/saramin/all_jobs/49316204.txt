서버 개발(백엔드)
서비스 소개
스왈라비(주)는 삼성전자 사내벤쳐에서 출발한 스타트업입니다.
커뮤니티형 건강관리 플랫폼을 개발하고 있습니다.
이 포지션의 AI 예상 면접 질문이 궁금하다면 점핏에서 확인해보세요!
모집부문 / 상세내용
사용 기술
Python Git Linux GCP Amazon S3 Amazon RDS Amazon EC2 AWS Java
주요업무
• AWS 환경의 개발 및 운영
• 서비스 API 개발
• 어드민(Admin) 서버 개발 및 운영
• 버그 원인 파악 및 문제 해결
자격요건
• 다양한 직군의 사람들과의 원활한 커뮤니케이션이 가능한 분
• Data Structure / Algorithm의 이해가 있는 분
• OOP(Object Oriented Programming)에 대한 이해가 있는 분
• Java 혹은 Python 기반 개발 경험이 있는 분
• Linux OS 에 대한 기본 지식이 있는 분
• Git 기반 협업 경험이 있는 분
• 인프라 : AWS, EC2, RDS, S3, GCP Infra
우대사항
• AWS를 활용한 개발 환경 인프라 구축 및 운영 경험이 있는 분 
• Java/Spring Boot 환경에서의 개발 경험이 있는 분
• DB 쿼리 작성 및 개선이 가능한 분 
• CI/CD 와 DevOps에 관심 있는 분
• 빅데이터 분석 경험이 있거나 관심이 있는 분
• 트래픽 많은 서비스 개발, 운영, 개선 경험 있는 분
• 컴퓨터공학 관련 학위 소지자 
• 애자일 프로세스에 대한 경험
마감일 및 근무지
• 마감일 : ~2024-12-01
• 근무지
- 서울특별시 강남구 테헤란로2길 27 1220호(역삼동, 패스트파이브빌딩)
복지 및 혜택
• 강남역 3분 거리 사무실
• 최신형 PC, 듀얼 모니터, 노트북, 기타 사무용품 지원
• 업무 필요 소프트웨어, 기타 협업툴(Google workspace, Jira, Confluence, Slack) 지원
• 유연 출근제
 - 정해진 출근 시간 없이 오전 8시 ~10시 사이 편하게 출근합니다. 
• 편안한 근무 환경
 - 2호선/신분당선 강남역 3분 거리 공유오피스에 위치하여, 커피/음료?/간식이 상시 제공되고 있습니다. 
• 자유로운 휴가
 - 연차/반차 휴가는 언제든 자유롭게 사용하세요. 
• 성장 지원
 - 업무와 관련된 책, 스터디, 강의 등은 적극적으로 지원해 드립니다. 
• 기타
 - 청년 소득세 감면 등 다양한 정부지원 프로그램에도 참여하고 있으니 편하게 요청해 주세요.
