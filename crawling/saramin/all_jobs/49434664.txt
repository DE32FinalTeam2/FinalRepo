Hardware Engineer 모집 포지션
■ NPU Architect                       ▶ 공고 보러가기
기본적인 모델부터 SOTA 모델까지 다양한 인공지능 모델을 NPU로 구동시킬 수 있는 원천기술을 개발합니다. ㈜딥엑스의 NPU HW Architecture는 고성능+저전력을 목표로하고 있으며, 이를 위해 종래의 NPU 에너지 효율성을 극대화시킬 수 있는 Data Workload를 개발하고, 사용자의 니즈와 NPU 스펙 간의 정합성과 타당성에 대해 지속적으로 연구합니다. CPU, Memory, Bus 등 Chip Architecture에 대한 지식을 보유하고, System C를 이용한 Architecture Level Modeling 경험과 PPA Estimation 경험을 갖추신 HW 엔지니어 후보자 중 고객이 보다 다양한 응용 제품을 개발하는데 견고한 기술적 기반을 제공하는 데 가치를 느끼고 열정이 있는 분들을 초대합니다.

■ RTL 설계 엔지니어 (HW IP)    ▶ 공고 보러가기
㈜딥엑스의 AI 하드웨어 프로세서 개발과 관련된 Hardware IP를 설계합니다. Linux 환경에서의 Verilog RTL 설계 경험, Simulation Model 및 Testbench 설계 경험, RTL Simulation 및 Verification 경험, C 기반 모델의 RTL IP 설계 및 검증 경험, python / perl / C / C++ 등을 원활하게 사용한 경험을 갖추신 HW 엔지니어 후보자 중 고객이 보다 다양한 응용 제품을 개발하는데 견고한 기술적 기반을 제공하는 데 가치를 느끼고 열정이 있는 분들을 초대합니다.

■ FPGA Platform 엔지니어        ▶ 공고 보러가기
㈜딥엑스의 AI 하드웨어 프로세서 개발과 관련된 FPA 플랫폼을 구축하고 프로토타입을 개발합니다. Verilog-HDL 기반의 RTL 설계 경험, Simulation Model 및 Testbench 설계 경험, RTL Simulation 및 Verification 경험, FPGA built in logic analyzer 사용 경험, python / perl / C / C++ 등을 원활하게 사용한 경험을 갖추신 HW 엔지니어 후보자 중 고객이 보다 다양한 응용 제품을 개발하는데 견고한 기술적 기반을 제공하는 데 가치를 느끼고 열정이 있는 분들을 초대합니다.

■ SoC ASIC 설계 엔지니어        ▶ 공고 보러가기
㈜딥엑스의 AI 하드웨어 프로세서 개발과 관련된 SoC ASIC을 설계합니다. Verilog 및 System Verliog를 이용한 RTL 설계 경험, ARM Core / AMBA bus 및 Interconnect를 이용한 SoC 설계 경험, SoC Top Integration 및 Verification 경험, Linux / shell / tcl / python / perl 등을 원활하게 사용한 경험을 갖추신 HW 엔지니어 후보자 중 고객이 보다 다양한 응용 제품을 개발하는데 견고한 기술적 기반을 제공하는 데 가치를 느끼고 열정이 있는 분들을 초대합니다.
