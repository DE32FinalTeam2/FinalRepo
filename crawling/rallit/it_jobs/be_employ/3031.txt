책트코리아, 어떤 곳인가요?
"CHeKT는 전 세계, 실물 보안업체(B2B)와 엔드유저(B2C)를 위한 서비스를 만듭니다."

CHeKT Team은 복잡한 old technology는 개선하고, 산업의 개념과 방식을 유지하면서, 새로운 기술(Cloud, AI, Camera Analytics…)을 접목했습니다. 그렇게 탄생한 Bridge 기기는 기존 보안시스템을 대체, 확장할 수 있는 서비스를 제공합니다. Bridge를 통해 보안업체와 엔드유저는 센서와 연동된 카메라, 혹은 영상분석 AI가 연동된 카메라를 통해 혁신적인 비디오 보안을 실천합니다.
🎯 책트는 설립 이후 매년 RMR(recurring monthly revenue) 2배 성장, 올해는 10월에 이미 5배 성장을 달성했어요
🎯 책트는 현재 5만대의 카메라가 등록되어 있고, 매달 최대 500대의 카메라가 설치되고 있어요
🎯 책트는 현재 초당 4-5백건의 API 호출이 발생해요
짧은 시간에 폭발적인 확장을 하고 있는 지금,
책트는 작은 팀으로 성공하고 싶습니다.
함께 도전하고, 실패도 하지만, 같이 회복하면서 성장할 팀원을 찾고 있습니다.

[ 책트의 주요 서비스 ]
- Dealer Portal : 보안 관제 센터 고객을 위한 통합 관리 B2B 솔루션
- Monitoring Portal : 보안 관제 센터의 실시간 보안 관제를 혁신하는 B2B 솔루션
- End User App/Web: 보안 관제를 위한 직관적인 B2C App/Web 솔루션

백엔드 개발자 (신입, 경력), 어떤 일을 하나요?
책트는 유저가 겪는 어려움을 해결하기 위해 아래의 업무를 수행합니다.
유저의 입장에서 새로운 경험을 고민하고 기획합니다.

주요업무
API 개발 및 DB 설계: 새로운 기능 개발을 위해 API 생성, 데이터베이스 설계, 스펙 논의, 버그 수정 등의 백엔드 개발 업무를 담당합니다.
실시간 통신 기술 활용: Web RTC와 MQTT 등 다양한 실시간 통신 기술을 사용하여 고성능 서비스를 지원합니다.
서버 운영 및 관리: AWS, GCP를 활용하여 대규모 트래픽을 처리하고 로그 관리를 효율적으로 수행합니다.
K8s 기반 서버 관리: Kubernetes(K8s)를 도입해 서버 관리를 유동적으로 전환하고 있으며, 이를 통해 서버 운영의 효율성을 극대화하고 있습니다.
DB 분리 및 로그 관리 최적화: 폭발적으로 증가하는 트래픽에 대응하여 DB 분리와 효율적인 로그 관리에 중점을 두고 있습니다.
레거시 코드 개선: 기존의 레거시 코드를 정비하고, NestJS로 시스템을 개선해 나가고 있습니다.
