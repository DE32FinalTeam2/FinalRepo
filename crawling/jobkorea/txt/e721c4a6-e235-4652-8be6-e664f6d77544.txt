빅데이터 기반 강력범죄지도 구축사업 
추가 기능 개발(상주)
㈜클라모스는 2017년에 설립된 회사로 자본금 7억원, 매출액 101억, 사원수 50명 규모의 고용노동부에서 선정한 서울형 강소기업입니다. 서울 구로구 디지털로 242 (구로동, 한화비즈메트로1차)에 위치하고 있으며, 컴퓨터관련(SI포함) 서비스 사업, 패키지 S/W개발/공급 사업을 하고 있습니다.
포지션 및 자격요건
풀스택 개발
( 1명 )
담당업무
- 법무부 빅데이터기반 강력범죄지도 구축사업(기 구축완료) → 추가 기능 개발
- 판결문을 분석하여 범죄가 발생한 위치를 지도에 표시하는 사업입니다.

특이사항
- 상주 필수( 소스 코드 반출 불가한 상황 )
- 상주 위치 : 휘경동 법무부 위치추적중앙관제센터
* 상주 일정은 협의하여 결정 가능 ( 단, 작업은 모두 상주하여 진행 필요 )
* 개발 완료 시, 프로젝트 조기 종료 가능합니다. (기간 단위 계약X)

필요요소
- Python 개발
- Java, React(ts) 기반 web application 개발

해결해야는 과제
1) Python 로직
- OCR 및 NLP API 규격서 작성
- Python 서비스 API 정리 및 문서화
- Python 판결문 분석 로직에서 정규식으로 주소 추출

2) JAVA, React(Typescript)
- 서비스 관련 로직 (JAVA, React(Typescript))
- 판결문 중복 카운트 로직 검수 및 수정
- 동일 판결문 재 분석시 중복으로 데이터가 쌓이는 문제
- 지도에 데이터 표시 오류 수정
    ex) 강원특별자치도 이후 들어가면 아무것도 나타나지 않음 오류 수정
- 배포 및 테스트, 인수인계

산출물
- 소스 코드 원본

참고 사이트/앱
- 법무부 빅데이터기반 강력범죄지도 구축사업(기 구축완료)되었으며, 당사 개발자들을 통한 인수인계가능
